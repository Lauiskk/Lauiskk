## 🌟 Featured Project: Cyberpunk Portfolio

</div>

A futuristic, **cyberpunk-themed portfolio** featuring:

- 🖥️ **Interactive Terminal Emulator** - Fully functional CLI with custom commands (`help`, `about`, `skills`, `projects`, `matrix`)
- 🔗 **GitHub API Integration** - Real-time project fetching and dynamic statistics
- 🎨 **Cyberpunk Design** - Neon aesthetics with dark/light mode toggle
- ⚡ **Particle.js Animations** - Dynamic background effects and smooth scroll reveals
- 📱 **Fully Responsive** - Mobile-first design with custom cursor for desktop
- 🎮 **Interactive Elements** - Typing effects, animated skill bars, and counter animations

**Tech Stack:** `Vanilla JavaScript` • `HTML5` • `CSS3` • `GitHub API` • `Particle.js` • `EmailJS`
